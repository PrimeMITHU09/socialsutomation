Our Mission

We built Social Automation to reduce repetitive social publishing work. Instead of manually writing captions, preparing uploads, scheduling posts, and checking failures one by one, this dashboard brings those tasks into one workflow-driven interface.

Where real integrations are configured, the system can generate content with Gemini, stage media, queue posts, and publish to connected platforms. Features without live integrations should be treated as local UI tools, not guaranteed automation.

Why This Exists

Content creators, agencies, and businesses spend a lot of time on repetitive posting tasks. This project brings workflow automation, publishing controls, and API integrations into one place without requiring a heavy framework.

The Facebook Page and YouTube publishing paths use real API flows. Other areas of the product may still depend on local UI logic or starter/demo-style content and should be reviewed feature by feature.

How It Works

01

Connect Your Platforms

Link your Facebook Page and YouTube Channel via OAuth. Credentials are stored in local project data on your machine/server.

02

AI-Assisted Content

When Gemini is configured, the app can generate scripts, hashtags, SEO ideas, and analysis results. Without Gemini, those AI-backed features will not run.

03

Queue & Publish

Posts can be queued for scheduled publishing or published manually. The app applies small timing delays in some flows, but it cannot guarantee account safety or platform outcomes.

04

Track & Review

The dashboard shows queue state, workflow runs, and failures stored by this app. Backup/export features should be verified separately before relying on them operationally.

Frequently Asked Questions

Do I need programming experience?
Zero. The workflow builder is completely visual — drag, connect, configure. No code required.
Is this a real automation system?
Yes. It connects to real Facebook Graph API and YouTube Data API. When you set your credentials, posts go live to your actual pages and channels.
Will Facebook or YouTube ban my account?
No app can honestly guarantee that. This project uses official APIs where implemented, but your permissions, usage pattern, and platform policy compliance still matter.
Where is my data stored?
This app stores credentials, queue items, workflows, and run data in local project files on the machine/server running it. Optional backup/export flows should be tested by you before production use.
Do I need a Gemini API key?
Yes, for the Gemini-backed AI features. Without a Gemini API key, those AI endpoints return errors instead of using a fake local fallback.
Can I run this on my phone?
Yes. The interface is responsive and PWA-enabled. You can install it as an app and manage your automation from anywhere.
